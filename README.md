# Frutes App 🍎🥕

Frutes App is a Flutter-based fruit ordering application that provides a seamless shopping experience for fresh produce delivery.
The app is built using clean architecture concepts with Firebase integration for user authentication and data management.

## 🚀 Features

- 🔐 Authentication (Firebase Auth & Social Login)
  - Email/Password login
  - Facebook Authentication
- 🏠 Home screen with products & categories
- ⭐ Best Selling Fruits section
- ❤️ Favorite items management
- 🛒 Cart management
  - Add / remove items
  - Update quantities
  - Real-time cart updates
- 💳 Checkout & order confirmation
- 👤 User Profile management
- 🎨 Theme support (Light & Dark mode)
- 🌍 Multi-language support (Localization)
- ⏳ Proper loading & error handling

## 🧱 Architecture & Tech Stack

**Framework:** Flutter  
**Language:** Dart

**State Management:** BLoC (Business Logic Component)  
**Backend:** Firebase (Authentication, Firestore)  
**Social Auth:** Facebook Authentication  
**Local Storage:** Shared Preferences  
**Dependency Injection:** GetIt  
**Localization:** Flutter Localizations  
**UI:** Material Design widgets

**Clean separation of layers:**
- Presentation (Views & Cubits)
- Domain (Repositories & Use Cases)
- Data (Models & Data Sources)
